상품을 등록하기 앞서 상품에 대한 기본적인 정보를 입력합니다.

 

 

 

 

상품코드

 

 

상품의 코드는 10자리 숫자로 자동생성합니다. 직접 상품코드를 입력할 수도 있습니다.
상품코드는 영문자, 숫자, - 만 입력 가능합니다.

 

 

 

상품명

 

 

상품 이름을 입력합니다.

 

 

 

기본설명

 

 

상품명 하단에 상품에 대한 추가적인 설명이 필요한 경우에 입력합니다. HTML 입력도 가능합니다.

 

 

 

출력순서

 

 

숫자가 작을 수록 상위에 출력됩니다. 음수 입력도 가능하며 입력 가능 범위는 -2147483648 부터 2147483647 까지입니다. 입력하지 않으면 자동으로 출력됩니다.

 

 

 

상품유형

 

 

메인화면에 유형별로 출력할 때 사용합니다.
이곳에 체크하면 상품리스트에서 유형별로 정렬할 때 체크된 상품이 가장 먼저 출력됩니다.

 

 

 

 

 

제조사

 

 

상품의 제조사를 입력합니다. 입력하지 않으면 상품 상세페이지에 출력하지 않습니다.

 

 

 

원산지

 

 

상품의 원산지를 입력합니다. 입력하지 않으면 상품상세페이지에 출력하지 않습니다.

 

 

 

브랜드

 

 

상품의 브랜드를 입력합니다. 입력하지 않으면 상품상세페이지에 출력하지 않습니다.

 

 

 

모델

 

 

상품의 모델명을 입력합니다. 입력하지 않으면 상품상세페이지에 출력하지 않습니다.

 

 

 

 

 

상품명, 기본설명, 출력순서, 상품유형, 제조사, 원산지, 브랜드, 모델까지 입력한 후 예시로 어떻게 표시되는지 보도록 하겠습니다.

 

입력한 상품이 메인페이지에 출력되었습니다.

 

 

상품을 클릭해보겠습니다.

 

 

'기본정보'에서 입력 값이 위와 같이 입력된 것을 알 수 있습니다.

 

 

 

전화문의

 

 

전화문의에 체크했을 경우 상품 금액 대신 전화문의로 표시됩니다.

 

 

 

 

판매가능

 

 

잠시 판매를 중단하거나 재고가 없을 경우 체크를 해제해 놓으면 출력되지 않으며, 주문도 받지 않습니다.

판매가격과 바로구매, 장바구니 버튼이 사라진 것을 알 수 있습니다. 

 

 

 

 

쿠폰적용안함

 

 

설정에 체크하면 쿠폰 생성 때 상품 검색 결과에 노출되지 않습니다.

 

 

네이버쇼핑 상품 ID

 

 

네이버쇼핑에 입점한 경우 네이버쇼핑 상품ID를 입력하시면 네이버페이와 연동됩니다.
일부 쇼핑몰의 경우 네이버쇼핑 상품ID 대신 쇼핑몰 상품ID를 입력해야 하는 경우가 있습니다.
네이버페이 연동과정에서 이 부분에 대한 안내가 이뤄지니 안내받은 대로 값을 입력하시면 됩니다.

 

 

상품 번호는 '네이버쇼핑파트너존'에서 상품관리에 접속하여 등록하고자 하는 상품의 ID를 입력하면 됩니다.
일부 쇼핑몰의 경우 네이버쇼핑 상품ID 대신 쇼핑몰 상품ID를 입력하셔야 하는 경우가 있습니다. 네이버페이 연동과정에서 이 부분에 대한 안내가 이뤄지기 때문에 안내받은 대로 값을 입력하시면 됩니다.

 

 

상품설명

 

 

상품의 상세설명을 입력합니다.

 

 

작성한 상품설명을 확인하여보도록 하겠습니다.

 

 

상품정보에 입력한 상품 설명이 출력되었습니다.

 

 

 

모바일 상품설명

 

 

모바일에서 노출할 상품의 상세설명을 입력합니다. 
입력하지 않을 경우 상품설명이 모바일 화면에 노출됩니다.

 

 

작성한 상품설명을 확인하여보도록 하겠습니다.

 

 

하단 바로가기 링크인 DETAIL을 클릭합니다.

 

 

작성된 상품설명이 입력된 것을 확인할 수 있습니다.

 

 

 

판매자 e-mail

 

 

운영자와 실제 판매자가 다른 경우 실제 판매자의 e-mail을 입력하면, 상품 주문 시점을 기준으로 실제 판매자에게도 주문서를 발송합니다.

 

 

 

상점메모

 

 

상품의 사입처 등 상품과 관련된 메모를 입력합니다.
